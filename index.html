Subadots

code.fun.do Hackathon, Microsoft Academia Accelerator
download .ZIPdownload .TGZ

It is a Universal Windows App, a Strategic Two-Player Game developed in C Sharp and XAML.

This game is a modified version of the classic Dots game, Implemented by designing a Pentagonal Web inspired by Cairo Tiling (It's pretty awesome! check it out) and implementing Brute Force Algorithms to check for completed pentagons (1000 lines of bug-free code in 24-hours). Pretty Awesome huh?
